What it is all about

This website is the 1st project in the Udacity Front-End Web Development Nanodegree. The projects listed here don't exist at the moment.

Features

  • Bootstrap
    The layout is based on the Bootstrap (current version 4.0.0-alpha.6) grid system. Some class names are used for description or for future modifications. Not all of them are therefore present in the styles.css file.
  • Icons
    Font Awesome is used for all icons, including input group addons in the contact form.
  • Contact form
    The contact form is inspired by the front page example at Better Contact Form, but the code is mine.

What doesn't work

As I am building this website before moving on to learning JavaScript, only HTML and CSS are used here. Therefore some features don't work yet.

  • Contact form
    The Send button should send me an email with the subject, name, message and attachment provided by the sender. It currently doesn't do anything.
  • Search
    The search form in the navbar should allow the user to filter projects based on desired technology, tools or keywords.
  • Red buttons in modals
    Buttons in the Technologies and Tools section in modals should work the same way as the search form and provide a list of projects in which similar technologies are used.
